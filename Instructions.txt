Chemical Inventory application User Instructions

#Overview
This is a web-based application for managing a list of chemicals with functionalities for adding, editing, deleting, sorting, and rearranging rows. The system is designed using plain HTML, CSS, and JavaScript
 
#Features
- Add Row: Add a new chemical entry with default values.
- Edit Row: Modify the details of the selected chemical entry.
- Delete Row: Remove a chemical entry by specifying its ID.
- Sort: Sort columns in ascending or descending order by clicking on the headers.
- Move Up/Down: Navigate through the rows to select a specific entry for editing.
- Refresh: Reload the displayed data.
- Save: Log the current state of the inventory to the console.

 #Prerequisites
- A modern web browser (e.g., Chrome, Firefox, Safari).

#How to Use:

Add Row: Click the "Add Row" button to add a new row with default values. The row can then be edited.
Move Up/Down: Select a row and use the "Move Up" or "Move Down" buttons to rearrange it within the table.
Delete Row: To delete a row, click the "Delete Row" button, and a pop-up will prompt you to enter the ID of the row you want to delete.
Edit Row: Select a row and click the "Edit Row" button to make the row editable.
Save Data: Once all changes are made, click the "Save" button to save the table data locally.
Sort Table: Click on any table header to sort the column in ascending or descending order.
